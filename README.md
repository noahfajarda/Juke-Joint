# Project-2-social-spotify

# Usage

Go to localhost:3001
--routes:
----/artist == set artist
----/artist/:artist == variable artist
----album == set album
----/album/:album == variable album
----track == set track
----/track/:track == variable track

# Installation

npm i
npm i express
npm i dotenv
npm i express-session
npm i express-handlebars-hotreload
